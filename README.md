## Задание 1: sum_distance()
Функция суммирует все числа от `from_num` до `to_num` включительно

**Особенности:**
- Автоматически меняет аргументы местами, если первое число больше второго
- Работает с положительными и отрицательными числами

## Задание 2: trim_and_repeat()
Функция обрезает строку слева и повторяет её заданное количество раз.

**Параметры:**
- `string` — исходная строка
- `offset` — сколько символов обрезать слева
- `repetitions` — сколько раз повторить
